# DroidPath 🤖

An Android learning app for developers to track their Android/Kotlin journey.

## Tech Stack
- Kotlin
- Jetpack Compose
- Material 3
- Navigation Compose
- MVVM (coming soon)

## Features
- [ ] Login / Signup UI
- [ ] Navigation setup
- [ ] ViewModel + Flow 

## Setup
1. Clone the repo
2. Open in Android Studio
3. Run on emulator or device
